# 01 — Stratégie & analyse concurrentielle

## Le vrai angle business

Bornes n'est pas une app de running. C'est une **plateforme de conquête locale + activation magasin pour marques running**.

> Le Pokémon GO du running local, monétisé par les marques et les boutiques via du trafic physique en magasin.

## Positionnement

| | Virtuel ↔ Réel | Mondial ↔ Local | Austère ↔ Premium |
|---|---|---|---|
| Run An Empire | Virtuel | Local | Austère |
| Turf | Virtuel | Local | Austère |
| INTVL | Mi-réel | Local | Moyen |
| Strava | Réel (perf) | Mondial | Sérieux/élitiste |
| **Bornes** | **Réel (lots)** | **Hyper-local** | **Premium / viral** |

## Concurrents

### Directs

- **Run An Empire** — capture de lieux réels en marchant/courant, rivalité avec voisins, empire virtuel.
  - *Faiblesse exploitable :* orienté « jeu de stratégie virtuel », pas « culture running locale + récompenses réelles ».
- **Turf** — marche/course/vélo pour capturer des zones et gagner des points.
  - *Faiblesse exploitable :* généraliste, peu « running lifestyle », pas premium visuellement.
- **INTVL** — le plus dangereux. Positionnement « Run. Capture. Conquer. », batailles locales, classements.
  - *Faiblesse exploitable :* ancrage français/local, vrais quartiers, lots physiques, clubs parisiens, boutiques partenaires.

### Indirects

- **Strava** — réseau social dominant du sport, vend déjà des Sponsored Challenges aux marques.
  - *Danger :* peut copier les mécaniques si le marché grossit.
  - *Opportunité :* large, sérieux, parfois élitiste → Bornes plus fun, local, accessible, viral.
- **Pokémon GO / Niantic** — preuve que le jeu géolocalisé crée de la valeur et amène en lieu physique (Sponsored Locations). **Notre modèle Drive-to-Store n'est pas une fantaisie : il existe déjà dans le gaming géolocalisé.**

## La différenciation centrale

Pas « on capture des zones en courant » (déjà fait), mais :

> **On transforme les quartiers en compétitions running sponsorisées, avec des récompenses réelles récupérables en boutique.**

Quatre leviers vraiment différenciants :
1. **Les vrais lots** — dossards, chaussures, bons d'achat, accessoires, tests de foulée, réductions.
2. **Les quartiers réels** — on ne se bat pas pour « zone 4832 » mais pour Belleville, République, Montmartre.
3. **Le contenu partageable** — l'animation de fin de run (vague de couleur) = moteur viral.
4. **Le B2B local** — « je t'envoie des coureurs qualifiés en boutique, je mesure les scans QR ».

## Défensibilité

| Horizon | Défensibilité | Source du moat |
|---------|---------------|----------------|
| Lancement | Faible (6/10) | Tech seule = copiable |
| Maturité | Forte | Communautés verrouillées + partenaires exclusifs + données locales + marque |

> On ne prend pas le lead avec la tech. On le prend avec une **stratégie de distribution locale agressive** (voir [09-go-to-market](09-go-to-market.md)).

## Scoring réaliste (œil investisseur)

| Critère | Note | Analyse |
|---------|------|---------|
| Désir utilisateur | 8/10 | Running social en explosion, besoin de comparer/partager/appartenir |
| Différenciation | 7/10 | Forte mais pas unique sur la mécanique GPS |
| Scalabilité technique | 8/10 | H3, GPS, Mapbox, backend géospatial : faisable |
| Scalabilité business | 6,5/10 | Partenariats locaux lents mais industrialisables |
| Monétisation | 7/10 | B2B >> premium B2C |
| Défensibilité | 6/10 → forte | Faible si tech seule, forte si communautés+partenaires+data |

**Conclusion :** scalable **si** on construit un *playbook local réplicable* : Paris 10/11 → Paris entier → 92 → Lyon → Marseille → Londres.
